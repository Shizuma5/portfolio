研究

情報理論を用いた深層学習におけるネットワーク構造の自動最適化というテーマで研究を行なっています。 抽象度の高いテーマですがざっくりと説明をすると、深層学習モデルのネットワーク構造を層と出力との相互情報量を元に計算し、効率的に探索していく手法を開発する所謂Neural Architecture Search分野の研究です。

プログラミング言語

  • python ★★★ Djangoでのwebアプリケーション開発や、研究、kaggleで主に利用。最近主に使っている言語です。フレームワークが充実していて何をするにも便利な言語だなと感じています。

  • Ruby ★★★ Ruby on Railsでのwebアプリケーション開発で主に利用。サーバーサイドでの開発は主にRailsで行っています。まだ死んでないなと思っています。

  • Javascript ★★☆ webアプリケーション開発においてVue.jsでのフロントエンドに主に利用しています。最近使い始めましたがとても便利で学習コストが低いというのを実感しています。

  • CSS ★★☆ フロントエンドのデザインをする際に利用。最近勉強し始めましたが、わかってくると中々楽しいです。このサイトではBulmaというフレームワークを使っています。

  • C言語 ★☆☆ 授業で習った程度の実力ですが、ある程度読むことはできると思います。DXライブラリというライブラリを使ってシューティングゲームを作ったことはあります。

  • swift ★☆☆ iosアプリの開発に二回ほど携わった時に触りました。それっきりなので深い理解がないので触ったことがある程度です。

  • UWSC ★☆☆ windowsにて動かすことのできるOSマクロ言語です。シューティングゲームのシミュレーションを行う際に使った言語で、記法は少し特殊ですがキー入力やマウス操作なども扱うことができてとても便利です。

★の数は僕の中での書ける具合です。相対評価です。
